\doxysubsection{Detailed Description}
Estrutura para armazenar a execução de uma operação. 

Uma execução de uma operação contêm o id da operação(operation\+ID),o id da maquina(maquina\+ID),a unidade de tempo(usage\+Time) e contêm um apontador para a próxima execução de uma operação
